# World_Population_Dashboard
Using : Deepseek Ai  &amp; Power Bi 

## Features of this World Population Dashboard:
### Interactive Charts:
- Population trend line chart (1960-2023)
- Regional distribution pie chart
- Top 10 countries bar chart
- Population growth comparison line chart

### Filters and Controls:
- Region/continent selector
- Country search with autocomplete
- Year range slider (1960-2023)
- Update dashboard button

### Real-time Statistics:
- Current world population
- Annual growth rate
- Population density
- Selected year display

### Responsive Design:
- Works on desktop and mobile devices
- Clean, modern UI with gradient header
- Card-based layout for charts

### Data Handling:
- Mock data simulation (in a real app, you'd replace with actual API calls)
- Error handling with fallback to demo data
- Data formatting for large numbers
